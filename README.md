# Uni Halo 静态资源

## 说明
- docs：对应 `uni-halo-docs` 中的静态资源
- author：作者资源
- images：图片资源
- screenshots：应用截图资源
- videos：视频资源

## 使用方法

### github cdn

### gitee raw

#### 格式

```
https://gitee.com/uni-halo/uni-halo-static-resources/raw/main/{资源类型}/{资源名}
```

#### 示例

- 图片资源：`/images/xx.jpg`
- 应用截图：`/screenshots/xx.jpg`
- 视频资源：`/videos/xx.mp4`
